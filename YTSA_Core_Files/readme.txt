YTSA_Core_Files is an acting library for the project. 

Files included have functions for managing the database including SQL models referenced in the project. 

sql_admin_functions.py - Adnimistrative functions used to add info into the DB or for validation
sql_models.py - primary location for class definitions of all tables used in the project
sql_requests.py - General functions for retreiving and updating information in tables. May merge with admin_functions soon.

More will be added as needed
